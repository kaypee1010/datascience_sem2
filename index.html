What is Probability?
🎲

Probability = chance of an event happening.

It answers: "How likely is something to occur?"
THE FORMULA
P(E) = Favourable Outcomes / Total Outcomes
CERTAIN
P = 1
Will definitely happen
RANDOM
0–1
May or may not happen
IMPOSSIBLE

1. Probability Rules 📊 PROB
Addition & Conditional Rules
➕ Addition Rule
P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
Used when you want the probability of event A or event B happening.
Numerical
P(A) = 0.4, P(B) = 0.5, P(A∩B) = 0.2
P(A∪B) = 0.4 + 0.5 − 0.2 = 0.7
🔗 Conditional Probability
P(A | B) = P(A ∩ B) / P(B)
Probability of A given that B has already happened.
Real-life Example
P(B) = 0.65,  P(A∩B) = 0.30
P(A|B) = 0.30 / 0.65 ≈ 0.46
Probability of passing given attendance > 75%

2. Hypothesis Testing – Core Concepts 🧪 HYP
H₀ vs H₁ — The Two Hypotheses
CORE IDEA
We don't just "feel" something is true — we test it using data.
H₀ Null Hypothesis No Effect
"No change / No effect" — this is the default assumption we try to disprove.
Example
H₀: Extra classes do NOT improve marks
H₁ Alternative Hypothesis Has Effect
"There IS an effect" — what we are trying to prove with data.
Example
H₁: Extra classes improve marks
🪜 Steps of Hypothesis Testing
  • 1
    Define H₀ and H₁ clearly
  • 2
    Choose significance level α = 0.05
  • 3
    Collect data from sample
  • 4
    Calculate test statistic
  • 5
    Find p-value
  • 6
    Compare p with α → Make decision
  • 7
    Interpret the result in context
📖 Classroom Story
"Average marks = 50"
Is it really true? Or just a coincidence in one exam? That's exactly what hypothesis testing helps us find out! 🎯
2. p-value & Significance Level 🧪 HYP
The p-value — Your Decision Tool
🔬
What is p-value?
Probability that your observed result happened purely by chance, assuming H₀ is true.

Smaller p-value = stronger evidence against H₀
α — Significance Level
The threshold we set before the experiment.
Usually α = 0.05 (5%)
IF
p ≤ α
❌ Reject H₀
Result is statistically significant!
IF
p > α
✅ Fail to Reject H₀
Not enough evidence.

2. Type I & Type II Errors 🧪 HYP
Error Types — When We Get It Wrong
TYPE I ERROR
❌ False Positive
We reject H₀ when it was actually true.
We think something happened — but it didn't.
🏫 "We say classes improved results… but actually they didn't."
TYPE II ERROR
⚠️ False Negative
We fail to reject H₀ when it was actually false.
We miss a real effect.
🏫 "Classes actually worked… but we failed to detect it."
Reality \ Decision Reject H₀ Fail to Reject H₀
H₀ is TRUE Type I Error ❌ False positive Correct ✓
H₀ is FALSE Correct ✓ (Power) Type II Error ⚠️ False negative
🎯 Classroom Analogy — Catching a Cheater
✅ Correct
Student IS cheating → You say cheating
❌ Type I
Student is NOT cheating → You say cheating
⚠️ Type II
Student IS cheating → You say not cheating

4. Regression Analysis 🔮 REG
Regression — Predicting the Future
📐
What is Regression?
Regression finds the Line of Best Fit — a line that best describes the relationship between X and Y, so we can predict Y from any X value.
EQUATION OF LINE
y = mx + c
SLOPE (m)
m = (y₂−y₁)/(x₂−x₁)
How steep the line is
INTERCEPT (c)
Where line crosses Y-axis when X = 0
📖 Example from Notes
Hours: 2→50 marks, 4→70 marks
Slope m = (70−50)/(4−2) = 10
Equation: y = 10x + 30
5 hours → y = 10(5) + 30 = 80 marks! 🎯

5 & 6. Scatter Plot + Time Series 📅 TS
Time Series Analysis
📅
What is Time Series?
Data collected over time at regular intervals. Time is always on the X-axis.
1. Trend 📈
Long-term movement (upward/downward)
e.g., Sales growing year over year
2. Seasonality 🌊
Repeating pattern in fixed periods
e.g., Ice cream ↑ every summer
3. Irregular / Random ⚡
Unexpected, unpredictable fluctuations
e.g., COVID impact on sales
