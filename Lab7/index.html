3 types of problems highlighted above

1
Missing values

Riya's Marks = "—" (blank). Ananya's Study Hours = "—" (blank). You can't leave blanks in a dataset — it breaks averages and formulas.

2
Inconsistent formats

Gender column has: Male, male, M, F, Female — all meaning the same thing but written differently. Payment has: UPI, upi, Cash, cash. Excel treats these as totally different values.

3
Outliers (suspicious extreme values)

Rahul's Marks = 500 (impossible — max is 100). Kabir's Attendance = 110% (impossible — can't exceed 100%). These are data entry errors.

Why does this matter?

If you calculate average marks with 500 included, your answer is wildly wrong. Dirty data = wrong decisions.

Step-by-step: fixing the data in Excel

Step 1 — Fix missing values

Rule:

Missing Marks → replace with AVERAGE of other marks. Missing Study Hours → replace with MEDIAN of other study hours. Why median for hours? Median is not affected by extreme values.

Try it — calculate the replacement values

Known marks (excluding missing & outlier): 78, 65, 82, 76, 30, 89

Known study hours (excluding missing): 3, 4, 2, 1, 3, 0, 5

=AVERAGE(F2,F4,F5,F7,F8,F9)
Average = (78+65+82+76+30+89) ÷ 6 = 420 ÷ 6 = 70
So Riya's marks → 70
=MEDIAN(E2,E3,E4,E6,E8,E9,E10)
Sorted: 0, 1, 2, 3, 3, 4, 5 → middle value = 3
So Ananya's hours → 3

What is normalization and why do we need it?

Imagine comparing marks (out of 100) with study hours (0–5). They're on completely different scales. Normalization brings everything to a 0–1 scale so we can compare fairly.

The formula

Normalized Value = (X − Min) / (Max − Min)

Min marks (after cleaning) = 30 (Arjun)  |  Max marks = 89 (Kavya)

Practice — normalize any student's marks

Excel formula to type in the new column

Viva preparation — 7 questions with model answers

Q1. What is dirty data?
Dirty data is data that contains errors, missing values, duplicate entries, or inconsistent formats — making it unreliable for analysis.
Q2. Why is data cleaning important?
Because inaccurate data leads to wrong conclusions. Cleaning improves data quality and makes analysis trustworthy and meaningful.
Q3. What is normalization?
Normalization scales data to a 0–1 range using the formula: (X − Min) / (Max − Min). It allows fair comparison between values that are on different scales.
Q4. What is the difference between mean and median?
Mean is the average (sum ÷ count) — it is affected by outliers. Median is the middle value when sorted — it is not affected by outliers. That's why we use median for study hours when outliers may exist.
Q5. Why are outliers dangerous?
Outliers are extreme values that distort statistical results. In our dataset, Marks = 500 inflated the average from ~69 to ~163 — giving a completely false picture of class performance.
Q6. What is data storytelling?
Data storytelling is the process of converting raw data and analysis results into clear, meaningful insights and actionable decisions — using charts, summaries, and plain language that non-technical people can understand.
Q7. Why should data be standardized?
Inconsistent formats (like Male/male/M) make grouping and counting impossible. Standardization ensures data can be correctly filtered, sorted, and analyzed in tools like Excel or Python.
